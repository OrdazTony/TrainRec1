# TrainRec1
Here I alongside a group of students in my Capstone 2 class will be collaborating on a Semester-Long project named "TrainRec".

Trainrec is meant to work as a personal excercise/workout app that can be used for homeworkouts as well as is the gym. TrainRec is meant to provide a safe space for users to perform their workouts safely and effectively without the discomfort that can arise from 
being self-concious about one's phsysique. With TrainRec there is a set of trained digital eyes free of judgement that can aid users in their excercise journey by assessing form and keeping track of goals. 


Alongside this README file there is also a requirements.txt file that is meant to be used to download the dependencies needed
for TrainRec to function.
